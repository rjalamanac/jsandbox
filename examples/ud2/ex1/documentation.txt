Crea una página web que demuestre cómo utilizar la consola del navegador para registrar mensajes y cómo limpiar la consola. Debes tener dos botones: uno para registrar un mensaje en la consola y otro para limpiar la consola.

Utiliza HTML para crear la estructura de la página y JavaScript para manejar los eventos de clic de los botones. Cuando el botón "Registrar en la Consola" se haga clic, debe registrar un mensaje en la consola del navegador, y cuando el botón "Limpiar Consola" se haga clic, debe limpiar la consola.

Asegúrate de que los botones tengan IDs para que puedan ser referenciados desde JavaScript.
