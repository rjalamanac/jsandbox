Crea una página web que demuestre el proceso de depuración de código JavaScript. La página debe contener un botón "Calcular" y un párrafo para mostrar el resultado.

El código JavaScript adjunto contiene un error deliberado al intentar dividir un número por cero. Utiliza las herramientas de depuración del navegador para encontrar y corregir el error. Luego, muestra el resultado de la operación en el párrafo.

Asegúrate de que el botón y el párrafo tengan IDs para que puedan ser referenciados desde JavaScript.
