Crea una página web que demuestre cómo trabajar con objetos en JavaScript. Debes declarar un objeto llamado "persona" que tenga propiedades como nombre, edad, ciudad y si es estudiante.

Luego, accede a las propiedades del objeto y crea un mensaje que incluya la información de las propiedades. Muestra el mensaje en un párrafo en la página.

Utiliza HTML para crear la estructura de la página y JavaScript para definir el objeto y construir el mensaje. Asegúrate de que el párrafo tenga un ID para que pueda ser referenciado desde JavaScript.
