Crea una página web que contenga un botón y un párrafo. Cuando el botón se haga clic, el texto dentro del párrafo debe cambiar. 

Utiliza HTML para crear la estructura de la página, CSS para dar estilo al botón, y JavaScript para manejar el evento de clic del botón y cambiar el texto del párrafo.

Asegúrate de que el botón y el párrafo tengan IDs para que puedan ser referenciados desde JavaScript.
