Crea una página web que demuestre cómo trabajar con arrays en JavaScript. Debes declarar un array llamado "colores" que contenga varios colores.

Luego, realiza las siguientes operaciones en el array:
1. Agrega un nuevo color al final del array.
2. Elimina el segundo color del array.
3. Ordena el array alfabéticamente.

Crea un mensaje que incluya los colores después de realizar estas operaciones y muéstralo en un párrafo en la página.

Utiliza HTML para crear la estructura de la página y JavaScript para definir el array y realizar las operaciones. Asegúrate de que el párrafo tenga un ID para que pueda ser referenciado desde JavaScript.
