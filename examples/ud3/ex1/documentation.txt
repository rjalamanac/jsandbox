Crea una página web que demuestre el uso de variables y tipos de datos en JavaScript. Debes declarar variables para el nombre de una persona, su edad, si es estudiante o no, y una lista de colores favoritos.

Luego, crea un mensaje que incluya la información de las variables y muéstralo en un párrafo en la página. El mensaje debe ser algo como: "Hola, soy [nombre]. Tengo [edad] años. Soy estudiante/No soy estudiante. Mis colores favoritos son: [colores favoritos]."

Utiliza HTML para crear la estructura de la página y JavaScript para definir variables y construir el mensaje. Asegúrate de que el párrafo tenga un ID para que pueda ser referenciado desde JavaScript.
