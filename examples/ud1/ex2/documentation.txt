Crea un formulario de registro con campos para el nombre y el correo electrónico. La página debe validar que ambos campos estén completos antes de enviar el formulario. Si alguno de los campos no está completado, muestra un mensaje de error; de lo contrario, muestra un mensaje de éxito con los datos ingresados.

Utiliza HTML para crear la estructura del formulario, CSS para darle estilo, y JavaScript para validar el formulario y mostrar los mensajes. Asegúrate de que los campos del formulario tengan IDs para que puedan ser referenciados desde JavaScript.
